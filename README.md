# Shawarma Al Baik - Menu Digital

Application web responsive pour la gestion d'un menu de restauration rapide, permettant la sélection de produits et la transmission de commandes via l'API WhatsApp.

## Fonctionnalités

* **Menu Interactif :** Présentation structurée des sandwiches et desserts avec gestion dynamique des quantités.
* **Système de Panier :** Calcul automatique du montant total et mise à jour en temps réel des éléments sélectionnés.
* **Interface Flottante :** Indicateur de panier persistant permettant de visualiser le montant total durant la navigation.
* **Finalisation de Commande :** Génération automatique d'un message récapitulatif structuré pour l'envoi vers WhatsApp.
* **Design Professionnel :** Mise en page adaptative (Mobile-First) avec une typographie soignée.

## Technologies Utilisées

* **HTML5** : Structure sémantique du document.
* **CSS3** : Mise en forme avancée, animations de transition et mise en page via Flexbox et Grid.
* **JavaScript (Vanilla)** : Logique métier pour la gestion du panier et manipulation du DOM.
* **Google Fonts** : Intégration des polices Amiri et Poppins pour une expérience utilisateur optimale.

## Structure du Projet

Le code est organisé de manière modulaire pour faciliter la maintenance :

- `index.html` : Contenu et structure du site.
- `style.css` : Définitions graphiques et gestion du responsive.
- `script.js` : Fonctions interactives et traitement des données de commande.

## Installation et Utilisation

1. Téléchargez l'ensemble des fichiers source.
2. Placez les fichiers `index.html`, `style.css` et `script.js` dans un répertoire unique.
3. Exécutez le fichier `index.html` via un navigateur web moderne.
